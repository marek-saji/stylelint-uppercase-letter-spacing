stylelint-uppercase-letter-spacing
==================================

Stylelint plugin to enforce defining `letter-spacing` with
`text-transform: uppercase`.

This plugin has no options at all, so:

```
{
    "plugins": [
        "stylelint-uppercase-letter-spacing"
    ],
    "rules": {
        "marek-saji/uppercase-letter-spacing": true
    }
}
```
